# RibaPurifier

React + Vite + TypeScript starter with styled-components and a modern Saudi/Qatar tech inspired theme.

Quickstart:

1. npm install
2. npm run dev

The project includes:
- Theme in `src/styles/Theme.ts`
- Global styles with Lexend + IBM Plex Sans Arabic in `src/styles/GlobalStyles.ts`
- Glassmorphism Navbar in `src/components/Navbar.tsx`
- Layout wrapper in `src/components/Layout.tsx`
